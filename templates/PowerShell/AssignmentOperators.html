Assignment Operators

More Assignment Operators

In the last lesson, we learned that we assign data to variables with the equals assignment operator. This is one of several operators that can be used to assign values to variables.

Operators

Symbol Description
= Assigns data to a variable
+= Increases the value of the variable by the specified value, or appends the specified value to the existing value.
-= Decreases the value of the variable by the specified value.
*= Multiplies the value of the variable by the specified value or appends the specified value to the existing value.
/= Divides the value of a variable by the specified value.
%= Divides the value of a variable by the specified value and then assigns the remainder (modulus) to the variable.
++ Increases the value of a variable, assignable property, or array element by 1.
-- Decreases the value of a variable, assignable property, or array element by 1.

Demonstrations

+=

First we initialize the variable $a and assign it a value of 1. After this we use the += operator to add 5. If you're new to programming, it may be helpful to think of it as shorthand for $a = $a + 5. Notice that nothing is displayed in the PowerShell console during this operation. In order to see the change, we have to type the variable name again in the console.

We also can use this operator as a shorthand for concatenating strings. In this case, we must provide a string for the first argument and a string for the second argument or other types, such as numeric types, which can be converted into strings.

-=

Next we have -=. We start by initializing the variable $b and assigning it a value of 21. We then subtract 32 from $b and assign the difference to the variable $b.

++

In programming, it is so common to increment or decrement a variable, that most languages provide a shorthand for x += 1 and x -= 1. The syntax for incrementing a variable is ++.

--

The syntax for decrementing a variable is --

The placement of ++ and -- matters

With the increment and decrement operators, you have the option to place them before or after a variable in PowerShell. You will get different results depending on where you place these operators. If you place ++ before a variable, it is called a pre-increment operator and if you place ++ after the variable it is called a post-increment operator. If you use the pre-increment operator, PowerShell will increment the value of the variable and return that value. If you use the post-increment operator, PowerShell will increment the value of the variable, but return the value of the variable before the increment operation.

*=

The *= operator like the += operator can accept numeric types or stings for inputs. With numeric types, it multiplies a variable by the argument provided. When we evaluate this variable on the next on line, we see that is has a value of 30.

/=

Here we demonstrate the /= operator. We start by initializing the variable $d and assigning it a value of 15. We then use the /= operator to divide $d by 3 and assign the difference to $d. The result is 5.

%=

Last we have the %= operator. This one might be unintuitive if you've just learned how the modulus operator works. If we expand this operator like we did the += operator it becomes $e = $e % 16. Since $e has a value of 9, it is the same as saying $e = 9 % 16. 9 % 16 is 9, so we assign the value 9 to the variable $e.
