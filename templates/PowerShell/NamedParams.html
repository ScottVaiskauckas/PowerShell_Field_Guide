    Named Parameters

    There is another kind of parameter in PowerShell called a named parameter. Unlike positional parameters, named parameters use the name of an argument to determine which parameter a programmer intended to use. Positional parameters save you a few lines of code, but at the cost of clarity. I prefer named parameters most of the time in PowerShell, though that is not true with all scripting languages.
