Logical Operators

What is a logical operator?

We have now arrived at one of my favorite topics and that is logical operators. This is not because logical operators in and of themselves are exciting, but I really enjoy logic as an academic subject. Logical operators evaluate the logic of a statement, that is whether or not the conditions of a statement are true.

Logical Operators

Name Syntax
-and Logical AND
-or Logical OR
-xor Logical Exclusive OR
-not Logical NOT
! Logical NOT

With the exception of !, these operators are kind of like the comparison operators we encountered before in that the syntax used to represent them is very different from that of most C-family programming languages and Python. In this case, we are also trading a programming convention for convenience in the shell console.

Demonstrations

Below I have examples of these operators and how they work in the shell console. They return true if the condition is true and false if it is false.

And (-and)

And will return true when condition 1 is true and condition 2 is true, but will return false when either condition is false or if both conditions are false.

Inclusive Or (-or)

Or works a little bit differently in logic than it does in regular English conversation. When English speakers use the word "or" in common usage, they typically mean exclusive or and not inclusive or. However, in logic the opposite is true - when a logician uses the word "or" they mean inclusive or. With inclusive or, as long as one of the conditions is true, the expression will evaluate to true. If both of the conditions are false, the expression will evaluate to false. This means that if both conditions are true with inclusive or, the expression is true.

Exclusive Or (-xor)

With Exclusive or if either of the conditions is true, the expression is true, but if both of the conditions are true or if both of the conditions are false, the expression will be false.

Not (-not/!)

Not returns true when a condition is false and false when a condition is true. The syntax for this operator is both -not and !. Above, I demonstrate using -not with the -and operator. Below I demonstrate using ! with the -or operator.
