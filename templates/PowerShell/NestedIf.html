    Nested If statements

    It may seem intuitive that it is possible to place an if-then statement inside of the code block of an else statement. This is called a "nested if statement" in programming.

    Example 1

    Our first example demonstrates a simple nested if statement. We begin by initializing a variable named $Condition1 and assigning it the value $False. We then initialize a variable named $Condition2 and assign it the value $True. Next, on line 4, we enter our first conditional statement. We test $Condition1 and find it is false, so skip over the code in the first code block. We then enter the code block of our else statement. Inside of this codeblock on line 9, we enter our second conditional statement. This conditional statement is "nested" inside of the first conditional statement. We test $Condition2 and find it is true, so execute the code block following the if statement. The script skips over the code block following the else statement and exits.
