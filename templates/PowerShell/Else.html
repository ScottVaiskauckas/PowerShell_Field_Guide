Else statements

Sometimes when you are writing scripts you will want to execute a code block if a test condition is met and a different code block if that test condition is not met. Rather than forcing you to write an exhaustive list of if statements, PowerShell provides you with the else construct.

Example 1

In our first example, we start by initializing a variable named $Condition1 and a variable named $Condition2 and assign $Condition1 the value $True and assign $Condition2 the value $False. Next on line 4, we hit our first if statement and test to see if $Condition1 is true. It in fact is so the first code block executes (lines 5-7) and PowerShell displays the text in the console. PowerShell then exits out of this code block and skips over the code block following the Else statement (lines 8-10). Following this, on line 12, we hit our second conditional statement and test to see if $Condition2 is true. It is not so we skip over our first code block (lines 13-15) and execute the second code block.
